# AI Resume Builder 🚀

An AI-powered Resume Builder that helps users create professional, ATS-friendly resumes in minutes. The platform streamlines resume creation through an intuitive interface, AI-assisted content generation, and customizable templates.

## 🌟 Features

- AI-generated resume content suggestions
- User authentication and secure access
- Dynamic resume creation and editing
- Multiple resume sections management
- ATS-friendly resume formatting
- Real-time preview
- Responsive and modern UI
- Resume download/export functionality
- Database-backed data persistence

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend / CMS
- Strapi

### Authentication
- Clerk Authentication

### Database
- MySQL

### AI Integration
- Google Gemini API

## 🏗️ System Architecture

1. User signs in using Clerk Authentication.
2. Resume data is stored securely in the database.
3. Gemini API generates content suggestions based on user inputs.
4. Users customize and preview resumes in real time.
5. Final resumes can be exported for professional use.

## 🎯 Problem Statement

Creating a professional resume can be time-consuming and challenging, especially for students and early-career professionals. This project simplifies the process by leveraging AI to assist users in generating high-quality resume content while maintaining professional formatting standards.
